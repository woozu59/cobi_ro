javascript 용어 01

선언

기본 선언법, 변수와 상수

자바스크립트는 선언과 호출을 기본으로 두고있다. 숫자나 문자열 등 다양한 데이터에 이름을 지정해서 쉽고 반복적으로 사용할 수 있도록 하는 것이 변수와 상수다. Variable(뜻.가변적인)을 기원으로 해서 변하는 수를 선언할 때는 var라는 선언법을 사용하여 잘 이용 하고 있었다. 하지만 var의 hosting(변수의 선언문을 유효 범위의 최상단으로 끌어올리는 행위)이 최상급이다보니, var를 사용함에 있어 오류를 뱉어야 하는 부분에서 오류가 나오지 않는 불편함이 생기기 시작했다. 이러한 불편함 때문에 변수를 선언하는 부분은 디테일 화 하였고 let과 const가 추가되어 업데이트가 되었다.

const는 'constance(뜻.불변/항구성)'의 약자이며 변하지 않는 수, 상수를 선언하는 용이고, let은 허락하다는 의미로, 변수를 선언하는데 사용한다. var는 function-scoped(함수범위)이고, let, const는 block-scoped(블록범위)이다. var가 전역변수가 아닌이상 함수범위로 움직이다보니 선언오류가 생길 수 있다. 이를 해결하는 방법이 즉시실행함수(immediately-invoked function expression (or IIFE, pronounced "iffy")를 이용하여 function-scope인거 처럼 만들어 사용할수도 있다. 선언을 하기 위해 너무 신경써야 될게 많다보니, 요즘은 let과 const로 구분하여 사용하는 추세이다.

자료의 유형

속성 사용방법 설명
문자열
(string)		 var t = "세종에 살고 있는 홍길동";
숫자
(number)		 var a = 1; 정수, 음수, 소수점 모두 숫자 타입
불린
(boolean)		 var c = true; 참(true)과 거짓(false)으로 두 가지 값만 존재
객체
(object)		 
var hotel = {
    room: 40,
    booked: 21,
    roomType: ["트윈","더블","스위트"],
    
    check: function(){
        return this.room – this.booked;
    }

객체는 이름이 지정된 값(키-값의 쌍)의 모음을 나타낸다. 보통 객체의 일부로 선언된 변수를 프로퍼티(property)라고 한다. 자바스크립트 객체의 프로퍼티에는 함수, 배열, 다른 객체를 비롯해 어떤 타입의 데이터도 담을 수 있다. 객체(object)란 웹브라우저를 포함한 웹 문서의 모든 구성요소이다.
배열
(array)		 
var array1 = new Array("Hello",123,true);
var array1 = ["Hello",123,true];
array1[0] = "Hello";
array1[1] = 123;
array1[2] = true;
배열은 값의 순차 컬렉션을 나타내는데, 각 값은 0부터 시작하는 인덱스 숫자를 가지고 있다. 자바스크립트에서는 배열명 다음에 대괄호로 인덱스를 감싸는 식으로 배열에서 값을 가져올 수 있다.
함수
(function)		 
function getArea(width, height){
    return width * height;
    //매개변수는 함수 내에서 변수처럼 사용된다.
}
    
getArea(); //함수호출
함수는 실행 가능한 코드가 담긴 특별한 객체다. 함수는 여러 번 호출될 수 있고, 인자를 전달받을 수 있다.
null null은 값이 없음을 나타내는 특별한 자료형으로서 객체가 없음을 나타내는 특별한 값
undefined 값이 할당되지 않은 변수는 미정의 타입 존재하지 않는 값

변수

자바스크립트에서 변수는 값과 연관된 이름으로써, 변수를 이용하면 자바스크립트 프로그램에서 데이터를 저장하고 조작할 수 있다. 자바스크립트 변수는 어떤 자료형의 값도 담을 수 있으며, 한 변수를 다른 타입의 값으로 할당할 수 있다.

변수명 규칙

  • 문자, 달러기호($) 혹은 언더스코어(_)문자로 시작하며, 절대로 숫자로 시작하지 않는다.
  • 문자, 숫자, 달러기호($) 혹은 언더스코어(_)문자를 포함할 수 있다.
  • 대시(-)나 마침표(.)같은 기호는 사용할 수 없다.
  • 대/소문자를 구분한다.
  • 변수가 저장할 정보의 종류를 잘 표현하는 단어를 사용하자.
  • 변수의 이름에 여러 단어를 사용해야하는 경우에는 첫 번째 단어를 제외한 나머지 단어의 첫 글자는 대문자를 사용하자. 캐멀 케이스(Camel case) 표기법이라고 한다.
  • 미리 정해진 키워드나 예약어(자바스크립트가 미리 확보한 단어)는 사용할 수 없다.

산술 연산자

속성 예시 설명
덧셈(+) c = a + b;
t = "홍"+"길동";		 + 연산자는 숫자 피연산자를 더하거나 문자열 피연산자를 연결
곱셈(*) c = a * b; * 연산자는 두 피연산자를 곱한다.
나눗셈(/) c = a / b; / 연산자는 첫 번째 피연산자를 두 번째 피연산자로 나눈다.
나머지(%) c = a % b; % 연산자는 첫 번째 피연산자를 두 번째 피연산자로 나눈 나머지를 반환
증가(++) b = c++; c 값을 변수b에 전달 한 후 c를 1씩 증가
b = ++c; c를 1 증가시킨 후 변수b에 전달
감소(--) b = c--; c 값을 b변수에 전달 한 후 c를 1씩 감소
b = --c; c를 1 감소시킨 후 변수b에 전달

할당 연산자

속성 예시 설명
= a = b; b의 값을 a에 할당한다
+= a += b; a와 b를 더한 값을 a에 할당한다
-= a -= b; a에서 b를 뺀 값을 a에 할당한다
*= a *= b; a에서 b를 곱한 값을 a에 할당한다
/= a /= b; a에서 b를 나눈 값을 a에 할당한다
%= a %= b; a를 b로 나눈 나머지 값을 a에 할당한다

비교 연산자

속성 예시 설명
== 
if(a==5){
    //실행문
};
== 연산자는 두 값이 동등한지 판단
!= 
if(a!=5){
    //실행문
};
!= 연산자는 두 값이 서로 동등하지 않을 경우 거짓을 반환하며, 그 밖의 경우 참을 반환
< 
if(a<5){
    //실행문
};
a가 5보다 작음 true
> 
if(a>5){
    //실행문
};
a가 5보다 큼 true

if(a<=5){
    //실행문
};
a가 5보다 작거나 같음 true
>= 
if(a>=5){
    //실행문

a가 5보다 크거나 같음 true

논리 연산자

속성 예시 설명
AND(&&) 
if(a>5 && a<10){
    //실행문
};
&& 연산자는 두 피연산자가 모두 참인 경우에만 참을 반환한다. 두 피연산자 가운데 하나 또는 둘 모두가 거짓일 경우 거짓을 반환한다. a가 5보다 크고, 10보다 작을 때 실행문을 실행한다.
OR(||) 
if(a!=5 || a>10){
    //실행문
};
|| 연산자는 하나 또는 두 피연산자가 참일 경우 참을 반환한다. 두 피연산자가 모두 거짓일 경우 거짓을 반환한다. a가 5가 아니거가, 10보다 작을 때 실행문을 실행한다.
NOT(!) 
if(!a){
    //실행문
};
! 연산자는 피연산자의 불린(boolean)값을 반대로 바꾼다. a가 참이 아닐 때 실행문을 실행한다.
