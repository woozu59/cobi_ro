요소 탐색하기(셀렉터)

분류 셀렉터 예시 설명
기본 셀렉터 * $("*") 모든요소
element $("p") 지정된 요소 이름을 가진 모든 요소
#id $("#lastname") 지정된 id 특성 값을 가진 모든 요소들
.class $(".intro") 지정된 class 특성 값을 가진 모든 요소들
.class, .class $(".intro,.demo") 하나 이상의 셀렉터와 일치하는 모든 요소들
계층 탐색 parent descendant $("div p") 요소의 하위 요소들
parent > child $("div > p") 요소의 직접적인 자식 요소들
element + next $("div + p") 요소의 바로 다음에 따라오는 근접한 이웃요소
element ~ siblings $("div ~ p") 요소의 모든 이웃 요소들
기본 필터 :not(selector) $("input:not('.hot')") 셀렉터에 부합하는 요소를 제외한 나머지 요소들
:first $("p:first") 선택된 요소 중 첫 번째 요소
:last $("p:last") 선택된 요소 중 마지막 요소
:even $("tr:even") 선택된 요소 중 인덱스 번호가 짝수인 요소들
:odd $("tr:odd") 선택된 요소 중 인덱스 번호가 홀수인 요소들
:eq(index) $("ul li:eq(3)") 선택된 요소 중 매개변수로 지정된 인텍스 번호를 가진 요소
:gt(index) $("ul li:gt(3)") 선택된 요소 중 매개변수로 지정된 인덱스 번호보다 큰 인덱스 번호를 가진 요소들
:lt(index) $("ul li:lt(3)") 선택된 요소 중 매개변수로 지정된 인덱스 번호보다 작은 인데스 번호를 가진 요소들
:header $(":header") <h1>부터 <h6>까지의 모든 제목 요소들
:animated $(":animated") 현재 애니메이션이 적용된 모든 요소들
:focus $(":focus") 현재 포커스를 가지고 있는 요소
콘텐츠 필터 :contains(text) $(":contains('Hello')") 매개변수로 지정된 텍스트를 가지고 있는 요소들
:empty $(":empty") 자식 요소가 없는 모든 요소들
:parent $(":parent") 자식 요소(텍스트 혹은 다른 요소)를 가지고 있는 모든 요소들
:has(selector) $("div:has(p)") 선택된 요소 중 매개변수에 지정된 셀럭터에 부합하는 요소를 최소한 하나 이상 가지고 있는 모든 요소(예를 들어, div:ha(p)는 <p> 요소를 가진 모든 div 요소를 의미한다.)
가시성 필터 :hidden $("p:hidden") 화면에 숨겨진 모든 요소들
:visible $("table:visible") 페이지의 레이아웃에서 공간을 차지하고 있는 모든 요소들
선택되지 않는 요소들: display: none; width/height: 0;인 조상 노드가 숨겨진 요소들
선택되는 요소들: visivility: hidden 또는 opacity: 0;인 요소들
자식 요소 필터 :nth-child(n) $("p:nth-child(2)") 1부터 시작하는 해당 순번의 요소
:first-child $("p:first-child") 현재 선택된 요소의 첫번째 자식 요소
:last-child $("p:last-child") 현재 선택된 요소의 마지막 자식 요소
:only-child $("p:only-child") 요소의 자식 요소가 하나뿐인 요소
특성 필터 [attribute] $("[href]") 지정된 특성을 가지고 있는 요소들(특성 값은 무관)
[attribute=value] $("[href='default.htm']") 지정된 특성값을 가진 요소들
[attribute!=value] $("[href!='default.htm']") 지정된 특성값이 지정된 값이 아닌 요소들
[attribute^=value] $("[title^='Tom']") 지정된 특성값이 지정된 값으로 시작하는 요소들
[attribute$=value] $("[href$='.jpg']") 지정된 특성값이 지정된 값으로 끝나는 요소들
[attribute*=value] $("[title*='hello']") 지정된 특성값이 지정된 값을 포함하고 있는 요소들
[attribute|=value] $("[title|='Tomorrow']") 지정된 특성값이 지정된 값과 일치하거나 지정된 값에 하이픈을 덧붙인 값으로 시작하는 요소들
[attribute~=value] $("[title~='hello']") 지정된 특성값이 공백으로 나열한 값 중 하나인 요소들
[attribute][attribute2] $("[href][title]") 지정된 특성중 하나를 가진 모든 요소들
폼 :input $(":input") 모든 입력 요소들
:text $(":text") 모든 텍스트 입력 요소들
:password $(":password") 모든 비밀번호 입력 요소들
:radio $(":radio") 모든 라디오 버튼 요소들
:checkbox $(":checkbox") 모든 체크박스 요소들
:submit $(":submit") 모든 전송 버튼 요소들
:image $(":image") 모든 <img>요소들
:reset $(":reset") 모든 재설정 버튼 요소들
:button $(":button") 모든 <button>요소들
:file $(":file") 모든 파일 요소들
:selected $(":selected") 드롭다운 리스트에서 선택된 모든 요소들
:enabled $(":enabled") 현재 활성화된 폼 요소들(기본적으로 모든 폼 요소들)
:disabled $(":disabled") 비활성화 된 모든 폼 요소들(CSS의 disabled 속성을 사용한 요소)
:checked $(":checked") 모든 선택된 라디오 버튼이나 체크박스 요소들
