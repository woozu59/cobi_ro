javascript의 개념과 이용하기

자바스크립트는 세계에서 가장 널리 사용되는 제일 가벼운 프로그래밍 언어이다. 일반적인 용도는 웹 페이지에 기능을 더해 HTML을 동적이고 살아 있게 만드는 것이다. Node.js, Apache CouchDB, Adobe Acrobat처럼 많은 비 브라우저 환경에서도 사용하고 있다. 자바스크립트로 할 수 있는 것은,

  • 콘텐츠에 접근하기

  • 콘텐츠 수정하기

  • 규칙을 프로그램하기

  • 이벤트에 반응하기

이다.

스크립트는 컴퓨터가 단계별로 따라 할 수 있는 명령들로 구성된다. 브라우저는 사용자가 웹 페이지와 어떻게 상호 작용하느냐에 따라 스크립트에서 필요한 부분만을 사용한다. 스크립트를 작성하려면, 아래의 과정을 거쳐야 한다.

  1. 목표를 설정한다

  2. 스크립트를 디자인한다: 작업 분류하기

  3. 스크립트를 디자인한다: 단계 구상하기

  4. 각 단계를 코드로 작성한다

예를 들어, 아크릴 조각 업체에서 명함제작 비용 계산을 해야하는 상황이다. 사용자가 입력한 글자 수만큼 비용을 지불하도록 계산 시스템을 만들고 싶다(목표 설정). 고객은 입력창에 본인이 원하는 이름을 입력할 수 있다. 비용은 한 글자당 5천원이고, 사용자가 이름을 입력하면 얼마를 지불해야 할 지 보여줘야 한다. 이 목적을 달성하기 위해 수행해야 할 작업들을 정의해보자(스크립트 디자인).

  1. 스크립트는 사용자가 버튼을 클릭할 때 실행된다

  2. 스크립트는 사용자가 양식에 입력한 이름을 알아낸다

  3. 사용자가 입력한 이름의 길이를 검사한다

  4. 사용자가 아무것도 입력하지 않으면 이름을 입력하라는 메시지를 출력한다

  5. 이름을 입력했다면, 입력된 글자의 길이에 글자당 가격을 곱하여 최종 비용을 계산한다

  6. 총 비용을 화면에 출력한다

이 작업 순서에 따른 흐름도를 그려보고 이해한 후 코드를 작성하면 된다.

현재 추세로는 HTML에 직접 작성하지 않고 스크립트 파일을 분리하여 연결을 하며, <head>나 <body>태그 안에 넣을 수 있다.
