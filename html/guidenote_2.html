Guide Note

Project Overview

러쉬는 자연에서 얻은 신선한 재료와 동물실험을 하지 않는 정직한 재료를 사용하여 모든 제품을 손으로 만드는 기업이며, 공정거래, 인권보호, 포장 최소화 등 다양한 캠페인 활동을 통해 기업윤리와 신념을 알리고 있습니다.


기존 사이트에서는 제품의 관련된 캠페인 및 재료 구성에 관한 정보가 흩어져 있어 제품의 다양한 정보를 찾는데에 어려움이 있었습니다.


‘채러피탓’ 제품에 관한 페이지를 원페이지 형식으로 제작하여 제품의 원재료 및 제품구매로 인한 후원단체정보 등을 한 눈에 볼 수 있으며 영상을 이용하여 페이지의 자연친화적인 분위기를 더해주었습니다.

Header

헤더 상단에는 러쉬공식홈페이지로 이동할 수 있도록 주소를 연결하였고 로고 아래에 제품명을 러쉬폰트를 이용해 작성하여 러쉬의 특유의 느낌이 느껴지도록 표현하였습니다.

Section1

Function

AOS

제품의 이미지들의 위치를 자유롭게 배치하였고 러쉬에서 추구하는 윤리이념을 상단에 배치하였습니다.


스크롤 시 제품의 주요 키워드가 차례대로 띄워지면서 기업의 이미지와 제품의 제작계기를 엿볼 수 있도록 하였으며, 자연풍경 영상을 작게 배치하여 자연친화적인 느낌을 표현하였습니다.

Section2

Function

AOS

제품의 원재료를 적당한 크기로 배치하고 순차적으로 떠오르게 하여 사용자에게 정보를 자연스럽게 전달할 수 있도록 하였습니다.

Section3

Function

Scroll Counting JS

Text Reveal

정해진 위치에 스크롤이 위치하였을 때 숫자가 카운팅 되도록 설정하여 단체에 후원한 총 금액을 표시하였습니다.


버튼을 클릭 시 후원한 단체의 리스트를 볼 수 있도록 주소를 연결하였습니다.

Section4

Function

Parallax

컨텐츠의 위아래로 margin을 10%로 주어 컨텐츠별로 겹치지 않도록 하였으며,


제품의 이미지에 parallax효과를 주어 컨텐츠에 시선이 갈 수 있도록 구성하였습니다.

Section5

Function

Swiper JS

제품의 관련된 다양한 아티클을 슬라이드 형식으로 제공하였고 autoplay로 다음 슬라이드로 넘어가도록 설정하였습니다.


하단에는 제품의 구매를 희망하는 사용자를 위해 제품명과 가격을 표시하였습니다.

Footer

Function

Scroll JS

기존의 러쉬 사이트의 컨셉을 유지하기 위해 기존의 footer과 동일하게 제작하였고 안의 불필요한 컨텐츠는 배제하였습니다.


scroll JS를 이용하여 TOP 이미지를 클릭 시 상단으로 부드럽게 올라갈 수 있도록 설정하였습니다.
