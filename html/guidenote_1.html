Guide Note

Project Overview

독도가 대한민국의 고유 영토인 역사적이고 지리적이며 국제법적인 근거와 이유를 제공하며, 독도의 자연경관을 엿볼 수 있도록 외교부에서 제작된 사이트입니다.


기존 페이지에서의 딱딱한 느낌과 사용자가 정보를 얻기 위해 복잡한 과정을 거치게 되는 불편함을 수정하고자 리뉴얼하여 사용자가 우선적으로 필요로 하는 정보를 더욱 한눈에 볼 수 있도록 하였으며 실시간의 독도의 날씨 및 온도를 알려주는 위젯을 통해 편의성을 높이고 독도의 자연경관을 영상 및 이미지를 통하여 생동감 있게 표현하였습니다.

Header & Section1

Function

Nav JS

Parallax

기존 사이트의 Nav는 관련 서브사이트가 뒤죽박죽 섞여있어 정보를 찾는데에 어려움이 있었습니다.


관련된 서브사이트를 하나로 묶어 새로 Nav를 구성하였고 js로 마우스를 갖다댈 시 자연스럽게 하강하도록 제작하였습니다.


기존의 딱딱한 느낌의 사진을 대신해 생동감 있는 독도의 모습을 보여주기 위해 독도 영상을 사용하였고 position을 fixed로 설정하여 Parallax효과를 주었습니다.

Section2

Function

openweather API

기존의 날씨 위젯은 작게 표시되어 있어 현재 독도의 날씨를 알고 싶어하는 사용자에게 보기에 불편함이 있어 크게 리뉴얼하였고 날씨 아이콘에 맞는 멘트로 날씨를 짐작할 수 있도록 설정하였습니다.


흩어져 있는 입도에 관련된 정보를 모아 아코디언 형식으로 정리하여 사용자가 날씨를 확인 후 입도의 정보를 얻을 수 있도록 배치하였습니다.

Section3

Function

AOS

독도에서만 볼 수 있는 자연경관을 카드형식으로 제작하여 aos를 이용하여 스크롤을 내릴 시 자연스럽게 떠오르며 나타날 수 있도록 설정하였습니다.
